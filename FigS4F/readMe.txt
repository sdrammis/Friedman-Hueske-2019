Light Aversion Info

BestBinCalculator found the time at which the mice spent largest percentage of time avoiding the light

PercentTimeInLightByIntensity calculated the percent time spent in light up to the optimal bin determined in BestBinCalculator for mice seperated by age and genotype

The data was complied in Percent Time In Light.xlsx and that table was imported into Prism (graphing software) to run a two way anova test
